1. PROJECT OVERVIEW
──────────────────
The Merit Scholarship Management System is an automated DBMS-backed portal 
designed to streamline scholarship eligibility validation, application tracking, 
and financial disbursement. It ensures transparency and efficiency in awarding 
merit-based scholarships to students.

2. DETAILED TECH STACK
─────────────────────
   FRONTEND:
   - Framework: React.js (v18.3.1)
   - Build Tool: Vite
   - Styling: Tailwind CSS & Vanilla CSS
   - Navigation: React Router DOM
   - Icons: Lucide-React
   - API Client: Axios

   BACKEND:
   - Language: Python (v3.10+)
   - Framework: FastAPI (Uvicorn server)
   - ORM: SQLAlchemy (v2.0+)
   - Data Validation: Pydantic
   - Auth: JWT (JSON Web Tokens) with Bcrypt hashing

   DATABASE:
   - System: PostgreSQL (Relational DBMS)
   - Driver: psycopg2

3. HARDWARE REQUIREMENTS (Minimum Recommended)
─────────────────────────────────────────────
   - Processor: Intel Core i3 (6th Gen) or AMD Ryzen 3 (or higher)
   - RAM: Minimum 4GB (8GB recommended for development)
   - Storage: 10GB free disk space (SSD preferred)
   - Network: Stable internet connection for API communication

4. SOFTWARE REQUIREMENTS
───────────────────────
   - Operating System: Windows 10/11, Linux (Ubuntu 20.04+), or macOS
   - Runtime Environments: 
     * Node.js (v18.x or higher)
     * Python (v3.10 - v3.12)
   - Database Server: PostgreSQL (v14 or higher)
   - Browser: Google Chrome, Mozilla Firefox, or Microsoft Edge (Latest versions)
   - Development Tools: VS Code, Postman (API Testing), Git

5. USAGE - WHERE AND HOW IT IS USED
──────────────────────────────────
   WHERE:
   - Universities and Autonomous Colleges
   - Government and Private Scholarship Boards
   - Financial Aid Departments

   HOW IT IS USED (Modules):
   
   A. STUDENT MODULE:
      - Account Creation: Students register and link their Student ID.
      - Eligibility Check: System automatically reads academic performance (CGPA/Rank) 
        and shows scholarships for which the student is "Eligible".
      - Application: Students apply for scholarships with a single click.
      - Tracking: Students trace the status from "Pending" to "Approved" and 
        finally "Disbursed".

   B. ADMIN MODULE:
      - Scholarship Management: Create and configure new scholarship programs 
        with specific CGPA and Rank cutoffs.
      - Verification: Review and approve/reject applications.
      - Disbursement: Track payments (Installment 1, Final Payout, etc.) 
        and update payment records in the database.
      - Data Management: Seed and sync academic records from college SLCM systems.
